# Eiendomsutvikling

Dette repoet er overordnet dashboard og integrasjonshub.

Mål:
- samle funksjonalitet fra flere forka repoer
- teste om systemene kan snakke sammen
- bygge et overordnet dashboard

Dette repoet eier:
- dashboard-frontend
- integrasjonslogikk
- dokumentasjon av dataflyt

Dette repoet eier ikke:
- all original logikk i de underliggende repoene
- intern kode som fortsatt skal bo i de enkelte forkene
