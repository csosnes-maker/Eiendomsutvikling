Dette repoet er et overordnet dashboard- og integrasjonslag.

Regler:
- Ikke flytt store mengder kode hit uten grunn.
- Behold underliggende forkede repoer som egne kodebaser.
- Foreslå først integrasjon via API, adaptere eller tydelige grensesnitt.
- Start med små, testbare steg.
- Når du foreslår endringer, skill mellom:
  1. hva som skal gjøres i dette repoet
  2. hva som eventuelt må gjøres i et underliggende repo
